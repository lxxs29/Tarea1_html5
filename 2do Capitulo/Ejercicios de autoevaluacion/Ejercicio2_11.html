Ejercicio 2.11

Indique cuál de los siguientes enunciados es verdadero y cuál es falso, explique por qué.

  1. a) Un documento de HTML5 válido no puede contener letras mayúsculas en el nombre de elemento.
    • Falso, HTML5 no distingue entre mayúsculas y minúsculas por lo que es igual la manera en como se escribe.

  2. b) Los documentos de HTML5 pueden tener la extensión de archivo .htm.
    • Verdadero, el formato htm es compatible totalmente.

  3. c) "&less;" es la referencia a entidad de carácter para el carácter menor que <
    • Falso, &less; no es una referencia valida.

  4. d) Es un documento de HTML5 válido, < li > puede anidarse dentro de etiquetas < ol > o < ul >
    • Verdadero, la etiqueta < li > es usada de forma anidada dentro de < ol > o < ul >
