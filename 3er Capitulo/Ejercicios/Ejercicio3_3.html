Ejercicio 3.3

Completa las siguientes oraciones.

  1. a) El atributo ___autofocus___ (que utiliza en un solo elemento input en un formulario) resalta de manera automática el elemento input y, si se usa de manera apropiada, coloca el cursor en el campo de texto después de que el navegador carga y despliega la página.

  2. b) Los nuevos tipos de input de HTML5 tienen ___Valicación automática___ del lado del cliente.

  3. c) El tipo input ___number___ permite al usuario introducir un valor numérico.

  4. d) El input ___range___ se valida automáticamente de manera intríca cuando lo despliega el navegador como un control deslizante, ya que el usuario no puede mover el control fuera de los límites del valor mínimo o máximo.

  5. e) El atributo ___autocomplete___ puede usarse en los tipos input para completar de manera automática la información del usuario con base en datos de entrada anteriores.

  6. f) El elemento ___figcaption___ proporciona una leyenda para la imagen en el elemento figure.

  7. g) El elemento summary muestra una flecha que apunta a la derecha enseguida de un resumen o leyenda cuando el documento se despliega en un navegador. Al hacer clic en la flecha, ésta apunta hacia abajp y revela el contenido del documento ___details___.

  8. h) El elemento mark nos permite ___resaltar texto___.
