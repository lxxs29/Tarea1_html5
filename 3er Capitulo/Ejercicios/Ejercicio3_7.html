Ejercicio 3.7

(Creación de un formulario con autocomplete). Cree un formulario de búsqueda. Use el navegador Web Firefox para probar el formulario. Escriba Junio y envíe el formulario. Después escriba una J en el elemento input para ver las entradas anteriores que empizan con J: deberá aparecer Junio debajo del elemento input. Escriba Julio y envíe el formulario de nuevo. Ahora escriba una J en el elemento input para ver las entradas anteriores que empiezan con J: deberán aparecer Junio y Julio debajo del elemento input. Prueb esto con sus propias consultas de búsqueda también.
