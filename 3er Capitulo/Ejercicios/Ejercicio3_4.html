Ejercicio 3.4

Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, exlpique por qué.


  1. a) Los navegadores que despliegan el tipo input color como campo de texto requieren que el usuario introduzca el nombre de un color.

    • Verdadero.

  2. b) Cuando un usuario introduce datos en un formulario y luego lo envía (por lo general, al hacer clic en el botón 'Enviar'), el navegador verifica de inmediato que los datos sean correctos.

    • Verdadero.

  3. c) HTML5 puede validar si la dirección de correo electrónico introducida por el usuario existe en realidad.

    • Falso, HTML5 no es capaz de detectar un correo electrónico falso, para ello se puede hacer con php. HTML5 solo puede verficar si la dirreción de email cumple con las formato adecuado.

  4. d) Puede agregar required a cualquiera de los tipos input.

    • Falso, required no puede ser usado cuando el tipo de input es: image, hidden, o un botón (reset, submit o button)

  5. e) Puede habilitar autocomplete sólo para ciertos elementos input.

    • Verdadero.

  6. f) El elemento time nos permite identificar una fecha, hora o ambas.

    • Falso, el elemento que correcto para esa acción es datetime.

  7. g) El elemento caption proporciona una leyenda para la imagen en un elemento figuere.

    • Falso, el elemento caption es utilizado en las tablas para proporcionar un titulo descriptivo.

  8. h) El elemento details muestra una flecha que apunta a la derecha enseguida de un resumen o leyenda cuando el documento se despliega en un navegador. Al hacer clic en la flecha, ésta apunta hacia abajo y revela el contenido del elemento summary.

    • Verdadero.

  9. i) El elemento footer describe el contenido por lo general aparece en la parte inferior del contenido o del elemento section.

    • Verdadero.

  10. j) El elemento highlight nos permite resaltar texto.

    • Falso, no es un elemento es una propiedad de CSS, para resaltar texto en HTML5 es necesario usar el elemento mark.
