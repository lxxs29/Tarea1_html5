Ejercicio 3.2

Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, esplique por qué.


  1. a) Cualquier tipo input particular de formulario de HTML5 debe desplegarse de manera idéntica en todos los navegadores compatibles con HTML5

    • Falso, cada navegador muestra de formas distintas los tipos input.

  2. b) Cuando se coloca el enfoque en el campo de texto (es decir, si el cursor está en el campo de texto), se envía el texto del atributo placeholder al servidor

    • Falso, el texto en placeholder solo es visible, no tiene un valor por lo cual no se env&ia.

  3. c) No es necesario incluir el atributo autofocus en sus formulario.

    • Verdadero.

  4. d) Los nuevos tipos input de HTML5 tienen validación automática del lado del cliente, con lo cual se elimina la necesidad de agregar secuencias de comandos (script) complicadas a sus formularios para validar la entrada del usuario y se reduce la cantidad de datos inválidos que se envían.

    • Verdadero.

  5. e) El tipo input range se valida automáticamente de manera intríseca cuando el navegador lo despliega como un control deslizante, ya que el usuario no puede mover el control fuera de los límites del valor mínimo o máximo.

    • Verdadero.

  6. f) HTML5 cuenta con validación automática para el tipo de input tel.

    • Falso, HTML5 no cuenta con una validación automática, principalmente porque la longitud de los números teléfonicos es variable en todo el mundo.

  7. g) Si el usuario introduce un URL con un formato inapropiado en un tipo input url, no se validará. HTML5 no valida si el URL introducido existe en realidad.

    • Verdadero.

  8. h) El elemento nav muestra un menú desplegable de hipervínculos.

    • Falso, el elemento nav sólo puede agrupar vínculos.

  9. i) El elemento header puede usarse sólo una vez en una página.

    • Falso, el elemento header puede ser usado varias veces en una misma página, en algunos casos se utiliza para encabazados o seccionar por partes el contenido de la página.

  10. j) Los elementos nav pueden anidarse en un elemento aside.

    • Verdadero.

  11. k) Podemos usar el elemento brk para evitar dividir palabras en lugares incorrectos.

    • Falso, el elemento correcto para este caso es wbr.
